  常见问题解答
什么是外汇？

外汇 (Forex，又称Fx) 是世界上最大及最具流动性的金融市场，它亦是一个分散的贸易市场，在银行、机构、投资者和个人之间，外汇的交易时刻进行着，平均每日交易量超过 5 万亿美元。与大多数的金融市场不同，外汇市场没有实际位置或中央交易所，而交易时间亦广至 7天24小时。

什么是汇率？

汇率是一国货币兑换另一国货币的比率，是以一种货币表示另一种货币的价格。由于世界各国货币的名称不同，币值不一样，所以一国货币对其他国家的货币要规定一个兑换率，即汇率。例如：于2015年12月31日，1 欧元可兑换 1.05 美元 ，于2016年11月7日，1 欧元可兑换1.11美元。在这段时间，欧元相对于美元的价值上升了约5.7%。部分货币的汇率是挂钩的，例如港元兑美元，这些货币的投资价值便会较低。

当我买卖外汇时，我究竟在做什么？

所谓的外汇交易，通常是描述投资者在外汇市场上同时买入某种货币和卖出另一种货币的情况。简单想象一下，假设美元相对于欧元的价值在未来预测会下跌。在这种情况下，投资者理论上会卖出美元和买入欧元，并待欧元的价值上涨到一定程度后，再买入美元，卖出欧元获取利润。这和投资股票的原理是相类似的，如果股票投资者认为股票的价格在未来会上升，便会买入股票，待价值上升后卖出。而不同之处，是外汇买卖可以进行双向交易，除了买入，还可以对货币对进行卖出的操作。

哪些因素驱动外汇市场？

外汇市场受许多不同因素的影响，例如政治和经济的稳定性、货币政策、干预外汇、天然灾害（地震，海啸......）。这些因素使外汇交易变得非常有趣及有吸引力。高流动性的市场让价格可以迅速改变，货币的价值会跟随其供应及需求的波动而波动。

•  某货币的供应增加或需求下跌可能会导致该货币的价值下跌。

•  某货币的供应下跌或需求增加可能会导致该货币的价值上涨。

为何要买卖外汇？
外汇交易从不歇息

由于外汇交易在世界各地的每一角落都能进行，因此，您可于每周5天，每天24小时随时进行操作，无论是白天或晚上都能交易。

双向交易双重机会

在外汇市场不存在熊市牛市的区别，无论涨跌都有获利机会。外汇市场不仅可以低买高卖，而且可以在您判断汇率会下跌的时候先卖出，汇率下跌后再买入平仓，只要选对交易方向均可获取盈利。

低交易成本

大多数外汇交易商不收取交易手续费，只通过点差（买入价格和卖出价格的差价）获利。国际提供的点差在外汇市场中位处行业低水平。具体收费可点击这里

杠杆500倍

投资者可以少量的资金控制较大的资产，提高客户的资金使用价值。不同产品杠杆也不同，目前外汇保证金交易的杠杆最大可达本金的500 倍，例如：投资 1000 美元就可以做 50 万美元的交易。需要注意的是，交易外汇中风险管理是非常重要，杠杆的高度使用会带来巨大收益同时也有机会导致巨额亏损。

透明公正不受操控

交易者遍及全球包括各国政府、银行、交易商及个人投资者，全球每日交易量超过 5 万亿美元，这么大的交易量不易被国家甚至个体任意操纵。同时外汇市场的行情和汇率可能随时受到及时发生的事件影响而变动，所以在外汇市场中不存在黑箱操作。

外汇交易元素
初始保证金

初始保证金是指客户下单时需要的保证金。

日内及周末强平保证金

日内强平保证金是指在正常交易时间内，需要保持的保证金水平，如果低于该水平将会被强制平仓。

周末强平保证金是指当客户持仓过周末的时候，需要保持的保证金水平，如果低于该水平将会被强制平仓。

点差

点差是买入价与卖出价之间的差价。买入价和卖出价的价差越小，对于投资者来说意味着成本越小。

长期交易下来，点差的大小对短线投资者的整体盈亏情况影响较大，而对中长线投资者几乎没有影响。

外汇交易实例

外汇报价只有两个价格：一个是卖价（又称买入价），另一个是买价（又称卖出价，有时称为要价）。这两个价格之间的差额便是点差。

计算方式如下：

总盈亏 = (卖出价 – 买入价) x 合约单位 x 交易手数 ± 过夜利息 (如有) – 佣金 (如有)

例一：

您通过差价合约平台进行交易：买入 0.5 手欧元/美元
（1手 = 100,000 欧元），买入价为 1.1050，并在即日以卖出价 1.1090 平仓。

总盈亏为：

(卖出价 – 买入价) x 合约单位 x 交易手数 ± 过夜利息 – 佣金
= (1.1090 - 1.1050) x 100,000 x 0.5 ± 0 - 0
= 200 美元

例二：

您通过差价合约平台进行交易：卖出 1 手欧元/美元
（1手= 100,000 欧元），卖出价为 1.1100，并在即日以买入价 1.1030 平仓。

总盈亏为：

(卖出价 – 买入价) x 合约单位 x 交易手数 ± 过夜利息 – 佣金
= (1.1100 – 1.1030) x 100,000 x 1 ± 0 - 0
= 700 美元
