  常见问题解答
什么是商品？

商品(英文称作Commodities)所包含的种类繁多，但在金融投资市场上，主要交易的为大宗商品。大宗商品指同质化、可交易及被广泛用作工业基础原材料的商品，主要分为3大类，即能源商品、基础原材料和农副产品，其中最为人熟悉的如原油、有色金属、钢铁、农产品、铁矿石、煤炭等都属于大宗商品。这几年来，全球对现存有限资源的关注日益高涨，其相对的投资价值也日渐增高，交易额也在逐年递增。例如黄金、原油等，其价格变动及相关的分析已成为市民大众的日常关注事项，许多投资者都已懂得制定自己的投资战略并通过我们的交易平台进行投资。想了解我们的商品交易成本?请点击这里。

什么是贵金属交易？

贵金属交易为目前在投资市场上最热门的产品之一。其交易是指投资者对贵金属市场看好的情况下赚取差价的过程，同时也可以是在经济前景被看淡的情况下所采取的一种避险手段,以实现资产的保值或增值。市场上最热门的贵金属投资产品可在进行交易，主要包括现货黄金（又称伦敦金）及现货白银（又称伦敦银），它们不是真正拿出实物黄金白银进行交收，而是透过一种保证金的方式进行交易。对于现货黄金，其每日交易量高达1万亿美元以上，故被称为世界上最大的股票，其交易遍布全球的主要市场如伦敦、苏黎世、纽约、芝加哥及香港等，因此没有任何机构及财团有足够的资金实力可以操控这个巨大的市场。

哪些因素影响到贵金属价格？ 美元

美元与黄金有着非常密切的关系，黄金是以美元计价的，而且在市场上，投资者一般会以黄金作为美元的替代投资产品，因此它们有着反向关系，比如美元价格上涨，黄金价格就会下跌。

政局动荡

政治事件也可能对黄金价格产生重大的影响。例如某些国家发生冲突事件，这一般会触发到投资者对该国债券或货币安全的忧虑，因此为规避风险，投资者在这个情况下多选择抽回资金以购买黄金。

供应与需求

如果黄金产量供应比需求高将会导致价格下跌。同样地，如果黄金产量需求增加而供应不足将导致价格上涨。

世界金融危机

当美国等西方大国的金融体系出现了不稳定的现象时，世界资金便会投向黄金，黄金需求增加，价格在大多数的情况下将上涨。黄金在这时就发挥了资金避险的功能。

什么是原油交易？

目前投资市场上所交易的最大原油产品分别有美国石油及英国石油。

美国石油又称西得克萨斯中间基原油，是产自美国内陆的一种轻质低硫原油，它具有良好的流动性及很高的价格透明度，是世界原油市场上的三大基准价格之一，同时以这种原油为基准油之交易成为了全球商品期货品种中的龙头。

英国石油又称布伦特原油，是指在北海生成的石油，其原油价格作在世界原油交易中同样有着重要的地位。

影响原油的主要价格 美元

原油价格历来都是和美元紧密挂钩的，但处于相反关系。比如美元若持续贬值，以美元标价的原油实际量将会下降，导致供应商必须通过提升原油价格作为应对措施来保持其价值的相对稳定。同样地，若美元升值，则油价会有所下调。

供应及需求

如原油产量供应比需求高将导致价格下跌。同样地，如果原油产量需求增加而供应不足将导致价格上涨。

经济及政治

尽管原油通常按期货价格进行交易，但是当前发生的经济事件却会影响这些价格波动。例如某个国家发生政治动荡多少也会影响到供应或需求导致价格波动。

其实商品交易是如何操作的?

大部分的商品是以期货合约来交易的。这些合约是您可以在一个指定的价格下对其进行买或卖的一个操作。就这样，您是不需要一个具体的空间去存放这些实物，例如几百桶的原油或者几千吨黄金等。

谁适合买卖贵金属及原油？

商品的投资者一般有着以下三种之一的特性﹕

套期保值者：由于商品的走势往往与其他投资产品的走势相反或者无关所以投资者经常买卖商品以帮助管理风险。

投机者：对特定商品持有看法的投资者愿意为获利而承担相应的风险。

经纪人：代投资者执行商品合约买卖指令的公司或个人。

为何要买卖商品？
杠杆最高约130倍

以杠杆方式放大投资金额，产品不同杠杆也不同，目前商品最高的杠杆约130倍。例如：投资100美元就可以做1万3千美元的交易。交易商品中风险管理是非常重要，杠杆的高度使用会带来巨大收益同时也有机会导致巨额亏损。

对冲功能

商品市场不存在熊市牛市的区别，投资者可买涨或买跌，无论涨跌只要选对方向均可获利。

套期保值

由于期货市场及现货市场是两个独立的市场，但是受同一的因素影响导致价格走势相同。因此投资者可分别在两个市场作相反方向的交易，以一个市场的盈利来弥补另一个市场的亏损，从而达到对冲。例如，如果现时的黄金价格下跌，黄金期货的合约就能起到增值的作用。

低成本交易

商品交易商基本上不收取任何交易费用，只通过点差获利。提供的点差在商品市场中位处行业最低水平，其中黄金低至0.26美元， 原油低至 0.036 具体收费可点击这里

商品交易元素
初始保证金

初始保证金是指客户下单时需要的保证金。

日内及周末强平保证金

日内强平保证金是指在正常交易时间内，需要保持的保证金水平，如果低于该水平将会被强制平仓。

周末强平保证金是指当客户持仓过周末的时候，需要保持的保证金水平，如果低于该水平将会被强制平仓。

点差

点差是买入价与卖出价之间的差价。买入价和卖出价的价差越小，对于投资者来说意味着成本越小。

长期交易下来，点差的大小对短线投资者的整体盈亏情况影响较大，而对中长线投资者几乎没有影响。

商品交易实例

商品报价只有两个价格：一个是卖价（又称买入价），另一个是买价（又称卖出价，有时称为要价）。这两个价格之间的差额便是点差。

计算方式如下：

总盈亏 = (卖出价 – 买入价) x 合约单位 x 交易手数 ± 过夜利息 (如有) – 佣金 (如有)

例一：

您通过差价合约平台进行交易：买入20手现货黄金
（1手=100美元），买入价为 1170.99，并在即日以卖出价1171.99 平仓。

总盈亏为：

(卖出价 – 买入价) x 合约单位 x 交易手数 ± 过夜利息 – 佣金
=（1171.99 - 1170.99）x 100 x 20 ± 0 - 0
= 2,000美元

例二：

您通过差价合约平台进行交易：卖出 5 手美国原油
（1手=1000美元），卖出价为52.945，并在即日以买入价51.445 平仓。

总盈亏为：

(卖出价 – 买入价) x 合约单位 x 交易手数 ± 过夜利息 – 佣金
=（52.945 - 51.445）x 1000 x 5 ± 0 - 0
= 7,500 美元
